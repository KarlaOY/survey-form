## 1. Intro

HTML5 trabaja de la mano con CSS3 para crear páginas web que usamos todos los dias en el navegador. Incluso, este sitio web en donde estás viendo este contenido esta construido con HTML y CSS. En este proyecto, con los conocimientos de HTML y CSS realicé un interfaz para un formulario de encuesta.

## 2. Qué construí

En este proyecto me enfoqué en construir un formulario para encuesta. Contiene las siguientes secciones:

* **Header**: Sección que involucra mi titulo y descripción del formulario.

* **Main**: Sección del contenedor para los elementos centrales de la página: inputs (text, email, number), selects, radios buttons, checkboxes, textarea y boton tipo sumbit.

## 3. Objetivo del Proyecto

Aprender a utilizar las etiquetas estandar de un forms con HTML5 y los estilos de CSS3.
